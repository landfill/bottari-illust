# 이미지 생성 프롬프트 템플릿

각 이미지는 따로 생성합니다. 본문 내용에 맞게 변수를 바꾸고, 여러 장을
한 이미지에 합치지 마세요.

```text
독립적인 16:9 가로형 한국어 본문 삽화 한 장을 생성하세요.

시각 DNA:
순백 배경. 단순한 검은색 손그림 선화. 약간 흔들리는 펜 선.
넉넉한 흰 여백. 오렌지색은 점무늬, 짧은 홍조, 작은 강조선처럼
소량만 사용. 조용하고 치유적인 여행 감성. 미니멀 라인아트.
독립 브랜드 로고 옆에 작게 놓을 수 있는 정돈된 그래픽 느낌.
그라데이션, 그림자, 종이 질감, 복잡한 배경, 상업용 벡터 스타일,
PPT 정보 그래픽 느낌, 어린이 일러스트, 과한 만화 표현,
사실적인 UI는 금지.

반복 등장 IP 캐릭터 필수:
보따리는 작고 귀여운 의인화된 여행용 배낭 캐릭터입니다.
전체적으로 작고 통통하며 둥근 실루엣입니다. 검은 손그림 선으로
앞주머니, 지퍼, 어깨끈, 작은 고리를 간결하게 표현합니다.
배낭 앞면에는 작은 검은 눈 두 개와 은은한 미소만 넣어
조용하고 친근한 표정을 만듭니다. 옆이나 뒤쪽에는 작고 둥근
천 보자기 꾸러미를 묶어 두고, 꾸러미에는 오렌지색 점무늬나
작은 포인트를 소량 넣습니다. 보따리는 무거운 마스코트가 아니라
핵심 개념을 조용히 운반하거나 정리하는 작은 여행 동반자입니다.

주제:
{본문 삽화 주제}

구조 유형:
{구조 유형: 워크플로 / 시스템 일부 / 전후 비교 / 역할 상태 /
개념 은유 / 방법의 층위 / 지도 경로 / 짧은 만화 컷}

핵심 의미:
{이 그림이 표현해야 하는 핵심 의미}

구도:
{구체적 화면: 보따리가 어디에 있고, 무엇을 조용히 운반하거나
정리하며, 주요 물건은 무엇이고, 정보는 어떻게 흐르는가}

보따리 자세:
{정면 / 왼쪽 3/4 측면 / 오른쪽 3/4 측면 / 살짝 숙인 자세 /
앉은 자세 / 3/4 후면 중 하나를 고르고, 이번 그림에서 보따리가
어떤 작은 동작을 하는지 쓴다. 이전 그림과 같은 방향과 자세를
반복하지 않는다.}

추천 요소:
{요소1} / {요소2} / {요소3} / {요소4}

한국어 손글씨 표기:
{표기어1} / {표기어2} / {표기어3} / {표기어4} / {선택 표기어5}

색 사용:
검정은 주 선화, 보따리 외곽선, 구조, 글자에 사용합니다.
오렌지는 보자기 점무늬, 작은 홍조, 핵심 이동 지점,
짧은 강조선에만 사용합니다. 빨강, 파랑, 초록 등 다른
포인트 컬러는 사용하지 마세요.

제약:
한 이미지는 하나의 핵심 구조만 설명합니다. 주 피사체는 캔버스의
약 40%-60% 안에 두세요. 최소 35%의 흰 여백을 보존하세요.
짧은 한국어 손글씨 표기는 최대 4-6개만 사용하세요.
왼쪽 위에 제목을 쓰지 마세요. 이미지 안에 구조 유형명을 쓰지 마세요.
공식 다이어그램, 강의 슬라이드, 빽빽한 설명 이미지처럼 만들지 마세요.
보따리를 검은 실루엣 괴물, 무표정 작업자, 어린이 만화 캐릭터,
스티커 마스코트처럼 만들지 마세요. 명확히 요청받지 않았다면
이전 예시를 복제하거나 알려진 사례 구도를 재사용하지 말고,
이 글에 맞는 새로운 시각 은유를 발명하세요. 여러 장을 만들 때는
보따리의 방향, 위치, 자세, 손동작을 반복하지 마세요. 분명하지만
설명서 같지 않고, 따뜻하지만 과장되지 않으며, 귀엽지만 절제되어야 합니다.
```

## 이미지 편집 프롬프트

왼쪽 위 제목 제거:

```text
제공된 이미지를 편집하세요. 왼쪽 위의 손글씨 제목
"{삭제할 글자}"와 밑줄만 제거하세요. 그 영역은 주변의 깨끗한
흰 배경과 자연스럽게 맞춰 채우세요. 캐릭터, 표기, 경로,
선 스타일, 구도, 화면비, 이미지 품질 등 다른 요소는 정확히
유지하세요. 새 글자나 물건을 추가하지 마세요.
```

보따리 일관성 강화:

```text
같은 핵심 의미와 단순한 레이아웃을 유지하되, 보따리를 첨부 기준의
작고 둥근 여행용 배낭 캐릭터로 다시 맞춰 생성하세요. 보따리에는
앞주머니, 지퍼, 어깨끈, 작은 고리, 옆이나 뒤쪽의 오렌지 점무늬
보자기 꾸러미가 보여야 합니다. 같은 시리즈의 다른 그림과 방향과
자세가 겹치지 않게 하되, 표정은 작은 눈 두 개와 은은한 미소만
사용하세요. 검은 실루엣 괴물, 무표정 작업자,
과장된 만화 마스코트처럼 보이면 안 됩니다. 깨끗하고,
여백이 많고, 손그림 같고, 조용하며 절제된 귀여움을 유지하세요.
```
