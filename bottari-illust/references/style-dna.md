# 스타일 DNA

## 한 문장

순백, 극도로 단순함, 손그림, 넉넉한 여백, 검은 라인아트, 아주 작은
오렌지 포인트, 조용하고 치유적인 여행 감성, 독립 브랜드 로고 옆에
놓을 수 있는 정돈된 본문 삽화.

AI, 제품, 디자인, 개발 도구를 오래 다뤄 온 사람이 흰 종이에 즉석에서
그린 설명 스케치처럼 보여야 합니다. 다만 반복 캐릭터 "보따리"는 검은
실루엣이 아니라 작고 둥근 여행용 배낭 캐릭터입니다.

## 반드시 지킬 것

- 16:9 가로형 본문 삽화.
- 순백 배경: 베이지색, 따뜻한 회색, 종이 질감, 그라데이션, 그림자, 노이즈, 복고풍 종이 느낌 금지.
- 검은색 손그림 선화를 중심으로 사용: 가는 선, 약간 흔들림, 기계적이지 않음, 벡터 느낌 아님, 두꺼운 외곽선 아님.
- 보따리는 작고 둥근 배낭 캐릭터로 표현: 앞주머니, 지퍼, 어깨끈, 작은 고리, 옆의 보자기 꾸러미를 단순하게 포함.
- 얼굴은 작은 눈 두 개와 은은한 미소만 사용.
- 넉넉한 여백: 주 피사체는 화면의 약 40%-60%, 최소 35%의 빈 공간, 가능하면 조용한 빈 영역 하나를 크게 둠.
- 소량의 한국어 손글씨 주석: 최대 4-6곳, 각 주석은 가능하면 2-8글자.
- 한 장은 하나의 핵심 동작, 구조, 상태 또는 은유만 설명.
- 구조는 자연스럽게 드러내고, 그림 위에 구조 유형명을 쓰지 않음.
- 전체 인상은 귀엽지만 절제되어야 하며, 유치한 아동 낙서처럼 보이면 안 됨.

## 색상

- 검정: 주 선화, 보따리 외곽선, 구조, 주요 글자, 핵심 물건.
- 오렌지: 보따리의 보자기 점무늬, 작은 홍조, 흐름의 핵심 포인트, 아주 짧은 강조선.

오렌지는 모든 그림에 넓게 쓰지 않습니다. 점, 작은 사선, 작은 표시처럼 적게 써야 따뜻함이 살아납니다.

## 절대 하지 말 것

- 상업 일러스트 금지.
- PPT 정보 그래픽 금지.
- 공식 흐름도 금지.
- 강의 자료 페이지 금지.
- 귀여운 만화 포스터 금지.
- 어린이 일러스트 금지.
- 복잡한 아키텍처 다이어그램 금지.
- 정교한 플랫 일러스트 금지.
- 기술적인 UI 느낌 금지.
- 실제 앱 스크린샷 금지.
- 복잡한 배경, 그라데이션, 그림자, 질감 금지.
- 고채도 색상이나 여러 포인트 컬러 금지.
- 빨강, 파랑, 초록 등 다색 주석 체계 금지.
- 모든 노드를 다 설명하려 하지 말 것.
- 왼쪽 위에 "워크플로 / 시스템 아키텍처 / 흔한 함정 / 로드맵" 같은 유형 제목을 쓰지 말 것.

## 미감 방향

치유계 손그림, 미니멀 라인아트, 귀엽지만 절제된 여행 캐릭터, 독립 브랜드 로고용 작은 그래픽에 가까워야 합니다.

차분하고, 산뜻하고, 생활감 있고, 단순하며 정돈되어야 합니다. 너무 괴상하거나, 음산하거나, 유치하거나, 복잡하거나, 딱딱하면 안 됩니다.
